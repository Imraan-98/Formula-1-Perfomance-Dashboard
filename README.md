🏎️ Formula 1 Performance Dashboard (1950–2023)
Interactive Business Intelligence dashboard built using Microsoft Power BI to analyse historical Formula 1 data, including driver performance, constructor trends, and race analysis.

📊 Overview
This project explores over 70 years of Formula 1 data, combining race results, drivers, constructors, and technical reliability to uncover patterns in competitive success and performance.

🔍 Key Metrics
Total Races: 1125
Total Drivers: 861
Total Wins: 1128
Technical Failures: 2480

📈 Key Insights
Formula 1 has expanded from a Europe-centric sport to a global competition
Ferrari, McLaren, and Williams show strong long-term dominance
Modern seasons show increased points due to format and system changes
Technical reliability plays a major role in race outcomes
